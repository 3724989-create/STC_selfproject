------------------------------重点提示，有关使用中断号大于31的中断服务函数或操作---------------------------------


1，使用中断扩充软件进行中断号扩充！！  ————非常推荐的做法！！！！

2，若是借用13号中断USER_VECTOR作为中断服务入口地址，务必确保isr.asm文件被添加到工程中，
   并在其中打开需要借用跳转的中断地址


详细方法参考：https://www.stcai.com/newsinfo/4342776.html


====================================LQ 8051U 资源分配=====================================


资源名称---------------------------分配内容----------------------------

---------------------4个硬件串口-------------------------------------------------------
UART1                              通用串口1(RX-P3.0 ,TX-P3.1)程序烧录,蓝牙
UART2                              通用串口2  (仅支持定时器2作为其波特率发生器)
UART3                              通用串口3
UART4                              通用串口4(RX-P0.2 ,TX-P0.3) GPS


---------------------6个定时器-------------------------------------------------------
Timer0                             通用电定时器0
Timer1                             通用电定时器1
Timer2                             串口波特率发生器
Timer3/4                           编码器采集(T3--> P0.4/P0.5; T4--> P0.6/P0.7)
Timer11                            通用电定时器11

--------------------- 两组PWM A/B---------------------------PWMA频率一致,PWMB频率一致!!!
PWMA_1(PWMA_ch1)                   舵机P1.0
PWMA_2                             舵机P1.2
PWMA_3                             舵机P1.4
PWMA_4                             舵机P1.6

PWMB_5(PWMB_ch1)                   电机机  PWM-P5.0            --气垫船中该管脚被用作串口3，接收SBUS遥控器使用
PWMB_6                             电机机  DIR-P5.1 (做普通I/O)
PWMB_7                             电机机  PWM-P5.2
PWMB_8                             电机机  DIR-P5.3 (做普通I/O)

---------------------1个通用I2C接口-------------------------
I2C_SCL                              P2.4
I2C_SDA                              P2.3

---------------------1个通用SPI接口------------------------- 默认给陀螺仪
SPI_CLK                              P4.3
SPI_MISO                             P4.2
SPI_MOSI                             P4.1
SPI_CS                               P4.0


---------------------1个屏幕-------------------------------- 默认使用软甲SPI(接口支持硬件SPI,但默认给陀螺仪)
IPS_SCL                              P3.2
IPS_SDA                              P3.4
IPS_RST                              P2.1
IPS_D/C                              P2.5
IPS_CS                               P2.6

---------------------传统GPIO外部中断-------------------------
INT脚位于P3口：INTO:P3.2,INT1:P3.3,INT2:P3.6,INT3:P3.7,INT4:P3.0

可用管脚:P3.3  P3.6, P3.7(剩余管脚用于其他外设占用)

---------------------按键/拨码开关-------------------------
KEY1                              P2.7
KEY2                              P4.5
KEY3                              P4.6
SW1                               P5.6
SW2                               P5.7
